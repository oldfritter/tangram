# 项目介绍

> [English](./README.md)

本项目是为了将分布在网络各处的服务整合成一个生产系统。

各服务节点之间通讯依赖MQ

# 流转过程

1. 用户访问WEB端页面

2. 页面发起数据查询请求至API节点

3. API节点包装请求数据，通过MQ发送给其它节点

4. 其它节点监听MQ，一旦收到数据立刻发送到对应的接口进行处理，并将处理后的数据通过MQ发送给API节点

5. API节点将从MQ收到的数据返回给WEB页面

# 项目文件结构

```
example           ----- 示例
lib/kafka         ----- 使用kafka作为通信的中间件
lib/rabbitmq      ----- 使用rabbitmq作为通信的中间件
lib/redis         ----- 使用redis作为通信的中间件
```
